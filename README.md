# Number Analyser

This is a python program that identify certain mathematical properties in number and generate them

## Features
--**Armstrong Numbers**
--**Automorphic Numbers**
--**Kaprekar Numbers**
--**Palindrome**

## How to Run:
1. Make Sure You have installed Python
2. Run the script 'python "NumberAnalyser.py"'
3. Using the menu to use the program
